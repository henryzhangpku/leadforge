## 🚀 Quick Start

```bash
git clone https://github.com/henryzhangpku/leadforge.git
cd leadforge
npm install
cp .env.example .env  # Add your Stripe keys
node src/server.js
```

Open `http://localhost:3000` — platform is live.

- **Backend:** Node.js, Express

- **Database:** SQLite (better-sqlite3)
- **Scraping:** Python, BeautifulSoup4
- **Payments:** Stripe-ready
